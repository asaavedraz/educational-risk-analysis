# Educational Risk Analysis

## Descripción

Proyecto de análisis de datos educativos para identificar factores asociados al riesgo académico de los estudiantes.

## Objetivos

* Analizar patrones de asistencia.
* Identificar variables relacionadas con el rendimiento académico.
* Crear visualizaciones de datos.
* Construir un modelo predictivo básico.

## Herramientas

* Python
* Pandas
* NumPy
* Matplotlib
* Scikit-Learn
* Power BI
